# changeablerevproxy

可通过HTTP网页进行反代端口修改。

```bash
➜  ~ ./changeablerevproxy -h
Usage: changeablerevproxy [OPTIONS] --target <TARGET>

Options:
  -p, --portal <PORTAL>  服务监听端口 [default: 9669]
  -t, --target <TARGET>  目标端口
  -h, --help             Print help
  -V, --version          Print version
```
